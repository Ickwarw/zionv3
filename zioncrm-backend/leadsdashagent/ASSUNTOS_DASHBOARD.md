## 🔌 COMO USAR NA API

### Listar todos os assuntos:
```bash
GET http://localhost:5004/api/dashboard/assuntos
```

### Listar assuntos de uma categoria específica:
```bash
GET http://localhost:5004/api/dashboard/assuntos?categoria=Classificação
```

### Listar todas as categorias:
```bash
GET http://localhost:5004/api/dashboard/categorias
```

### Obter dados de um assunto específico:
```bash
GET http://localhost:5004/api/dashboard/assuntos/perdas_clientes_fidelizados
```

## 📈 EXEMPLOS DE USO

### Criar gráfico combinando múltiplos assuntos:
```json
POST http://localhost:5004/api/dashboard/graficos
{
  "titulo": "Análise Completa de Vendas",
  "descricao": "Todos os indicadores de vendas em um único gráfico",
  "tipo_grafico": "line",
  "assuntos": [
    "taxa_conversao_vendas",
    "distribuicao_por_vendedor",
    "distribuicao_campanha",
    "leads_indicadas"
  ],
  "configuracoes": {
    "cores": ["#FF6384", "#36A2EB", "#FFCE56", "#4BC0C0"]
  }
}
```

### Filtrar assuntos por categoria no frontend:
```javascript
// Buscar todas as categorias
const categorias = await fetch('http://localhost:5004/api/dashboard/categorias')
  .then(r => r.json());

// Buscar assuntos de uma categoria específica
const assuntosVendas = await fetch('http://localhost:5004/api/dashboard/assuntos?categoria=Vendas')
  .then(r => r.json());

// Exibir no select dropdown
assuntosVendas.data.forEach(assunto => {
  const option = `<option value="${assunto.key}">${assunto.label}</option>`;
  document.getElementById('selectAssunto').innerHTML += option;
});
```

## ✨ RECURSOS AVANÇADOS

### 1. Combinação ilimitada de assuntos
- Combine quantos assuntos quiser em um único gráfico
- Misture assuntos de categorias diferentes
- Exemplo: Vendas + Localização + Demográfico no mesmo gráfico

### 2. Filtragem inteligente
- Filtre assuntos por categoria
- Busque por palavra-chave no label ou descrição
- Organize por relevância

### 3. Tipos de gráfico variados
- **bar**: Gráfico de barras verticais
- **line**: Gráfico de linhas (tendências temporais)
- **pie**: Gráfico de pizza (distribuições)
- **doughnut**: Gráfico de rosquinha
- **radar**: Gráfico radar (comparações multidimensionais)
- **polarArea**: Gráfico polar

### 4. Configurações personalizadas
```json
{
  "configuracoes": {
    "cores": ["#FF6384", "#36A2EB", "#FFCE56"],
    "legenda": {
      "posicao": "bottom",
      "exibir": true
    },
    "eixos": {
      "x": { "label": "Período" },
      "y": { "label": "Quantidade" }
    }
  }
}
```

## 💡 DICAS DE USO

1. **Comece simples**: Crie gráficos com 1-2 assuntos primeiro
2. **Agrupe por categoria**: Combine assuntos da mesma categoria
3. **Use cores diferentes**: Configure cores distintas para cada assunto
4. **Escolha o tipo certo**: 
   - Use **line** para tendências temporais
   - Use **pie/doughnut** para distribuições percentuais
   - Use **bar** para comparações diretas
   - Use **radar** para análises multidimensionais
5. **Salve seus favoritos**: Salve gráficos úteis para acesso rápido
